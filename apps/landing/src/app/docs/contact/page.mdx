# Contact

Have a question, found a bug, or want to suggest a feature? Here's how to reach us.

## GitHub Issues

The best way to report bugs or request features is through GitHub Issues:

<a href="https://github.com/monokaijs/block-based/issues" target="_blank" rel="noreferrer">
  github.com/monokaijs/block-based/issues
</a>

Please include:

- A clear description of the problem or feature request
- Steps to reproduce (for bugs)
- Your environment (browser, OS, package version)

## GitHub Discussions

For questions, ideas, or general conversation, use GitHub Discussions:

<a href="https://github.com/monokaijs/block-based/discussions" target="_blank" rel="noreferrer">
  github.com/monokaijs/block-based/discussions
</a>

## Contributing

We welcome contributions! To get started:

1. Fork the repository on [GitHub](https://github.com/monokaijs/block-based)
2. Create a feature branch from `main`
3. Make your changes and write tests if applicable
4. Open a pull request with a clear description
